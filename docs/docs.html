📚 Документация LiveM3U

🔧 Установка и запуск

Для запуска поискового робота выполните следующие команды:

# Клонирование репозитория
git clone https://github.com/your-repo/livem3u.git
cd livem3u

# Установка зависимостей
pip install -r requirements.txt

# Однократный запуск
python src/bot.py --once

# Непрерывный режим (обновление каждые 30 мин)
python src/bot.py --interval 1800

⚙️ Параметры запуска

  • --once — однократный запуск сканирования
  • --interval N — интервал повторного запуска в секундах (по умолчанию 1800 = 30 мин)
  • --ip-ranges — IP диапазоны для сканирования (опционально)

📁 Структура проекта

  • /src/bot.py — основной скрипт поискового робота
  • /config/sources.json — источники для поиска
  • /config/channels.json — конфигурация каналов
  • /data/playlist.m3u — сгенерированный плейлист
  • /data/found_streams.json — база найденных потоков
  • /data/channel_history.json — история каналов
  • /index.html — веб-интерфейс

🔍 Как работает поиск

Поисковой робот использует несколько методов для нахождения IPTV потоков:

  1. Сканирование публичных M3U плейлистов на GitHub и других ресурсах
  2. Поиск через поисковые системы (Google, Яндекс) по специальным запросам
  3. Анализ сайтов российских провайдеров и агрегаторов
  4. Поиск независимых телеканалов (Дождь, RTVI и др.)
  5. Проверка популярных IPTV паттернов URL

🌐 Прокси

Для обхода блокировок используется прокси secure-272717.tatnet.app. Каналы Zabava-HLive (ngenix.net) работают напрямую без прокси для максимальной скорости и стабильности.

📊 Формат плейлиста

Плейлист генерируется в формате M3U с метаданными:

#EXTM3U
#EXTINF:-1 tvg-name="Channel Name" group-title="Group",Channel Name
https://stream-url.com/playlist.m3u8

🔧 Настройка

Вы можете настроить параметры поиска в файле /src/bot.py:

  • IPTV_PORTS — порты для сканирования
  • RU_KEYWORDS — ключевые слова для поиска русскоязычных каналов
  • M3U_SOURCES — источники M3U плейлистов
  • ZABAVA_HLIVE_DIRECT — прямые ссылки на каналы Забава/Винк

1. Общие положения

1.1. Настоящие Условия использования (далее — «Условия») регулируют использование сервиса LiveM3U (далее — «Сервис»).

1.3. Сервис предоставляет автоматический поиск и агрегацию publicly доступных IPTV потоков из открытых источников интернета.

2. Описание сервиса

2.1. LiveM3U — это поисковой робот, который автоматически сканирует интернет на наличие publicly доступных IPTV потоков.

2.2. Сервис не хранит, не размещает и не транслирует видеоконтент. Все потоки находятся на сторонних серверах.

2.3. Сервис仅提供 ссылку на publicly доступные потоки, найденные в открытом доступе.

2.4. Плейлист обновляется каждые 30 минут для обеспечения актуальности информации.

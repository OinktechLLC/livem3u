🔒 Политика конфиденциальности

1. Общие положения

1.1. Настоящая Политика конфиденциальности (далее — «Политика») описывает, как LiveM3U (далее — «Сервис», «мы», «нас») обрабатывает информацию пользователей.

1.2. Мы стремимся защитить вашу конфиденциальность и обрабатываем данные в соответствии с Федеральным законом № 152-ФЗ «О персональных данных».

Важно: Сервис практически не собирает персональные данные пользователей.

2. Какие данные мы собираем

2.1. Данные, которые НЕ собираются:

  • Мы не собираем имена, фамилии, адреса электронной почты
  • Мы не собираем номера телефонов, паспортные данные
  • Мы не отслеживаем местоположение пользователей
  • Мы не используем cookies для отслеживания
  • Мы не сохраняем историю просмотров

2.2. Технические данные (автоматически):

  • IP-адрес (временно, для технической отладки)
  • Тип браузера и пользовательский агент
  • Дата и время доступа к сервису

3. Как мы используем данные

3.1. Технические данные используются исключительно для:

  • Обеспечения работоспособности сервиса
  • Устранения технических неполадок
  • Анализа нагрузки на сервер
  • Улучшения работы поискового робота

3.2. Данные не передаются третьим лицам, за исключением случаев, предусмотренных законодательством.

4. Логи поискового робота

4.1. Поисковой робот создаёт логи своей деятельности для:

  • Отладки и улучшения алгоритмов поиска
  • Мониторинга найденных потоков
  • Истории изменений плейлиста

4.2. Логи не содержат персональных данных пользователей и хранятся только на сервере.

5. Хранение данных

5.1. Технические данные хранятся не более 30 дней.

5.2. Логи поискового робота хранятся до момента перезапуска сервиса.

5.3. База найденных потоков (found_streams.json) содержит только информацию о URL каналов, без данных пользователей.

6. Защита данных

6.1. Мы принимаем технические меры для защиты информации:

  • HTTPS шифрование соединений
  • Регулярное обновление программного обеспечения
  • Ограничение доступа к серверам
  • Резервное копирование данных

7. Права пользователей

7.1. В соответствии с законодательством, вы имеете право:

  • Получить информацию об обработке ваших данных
  • Требовать уточнения, блокирования или уничтожения данных
  • Отозвать согласие на обработку данных

7.2. Поскольку Сервис практически не собирает персональные данные, реализация этих прав осуществляется автоматически.

8. Сторонние сервисы

8.1. Сервис может содержать ссылки на сторонние ресурсы (IPTV потоки, GitHub, прокси-серверы).

8.2. Мы не несём ответственности за политику конфиденциальности сторонних сайтов.

8.3. При переходе по ссылкам рекомендуется ознакомиться с политикой соответствующего сайта.

9. Изменения политики

9.1. Мы оставляем за собой право изменять настоящую Политику в любое время.

9.2. Актуальная версия всегда доступна по адресу /docs/privacy.html

9.3. Продолжение использования Сервиса означает согласие с изменениями.

10. Контакты

По вопросам, связанным с конфиденциальностью, обращайтесь через GitHub Issues проекта.

11. Применимое законодательство

11.1. Настоящая Политика регулируется законодательством Российской Федерации.

11.2. Для пользователей из ЕС применяется General Data Protection Regulation (GDPR).

11.3. Для пользователей из других стран применяются соответствующие местные законы о защите данных.

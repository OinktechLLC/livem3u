# FilmsTop - Платформа управления видеоконтентом

Проект переименован из **infokanal** в **FilmsTop**.

## Структура проекта

- `database.py` - Инициализация базы данных SQLite.
- `film_manager.py` - Управление фильмами (добавление, получение).
- `search_robot.py` - Заглушка для поискового робота (требует подключения легальных API).
- `generate_playlist.py` - Генерация `.m3u8` плейлиста с названием канала "LiveM3U Films".

## Установка и запуск

1. Инициализируйте базу данных:
   ```bash
   python database.py
   ```

2. Добавьте фильмы вручную (через Python или расширив код):
   ```python
   import film_manager
   film_manager.add_film("Название фильма", "Описание", "/путь/к/файлу.mp4")
   ```

3. Запустите генератор плейлиста:
   ```bash
   python generate_playlist.py
   ```

4. Полученный файл `stream.m3u8` можно открыть в VLC или другом плеере.

## Особенности

- **Логотип**: В плейлист добавлен тег `tvg-logo` со ссылкой на заглушку с текстом "LiveM3U Films". Для реального отображения логотипа поверх видео необходимо использовать серверное ПО (например, FFmpeg с фильтром overlay) или настроить плеер.
- **Поисковый робот**: В текущей версии не выполняет автоматический поиск в интернете. Вы можете доработать `search_robot.py`, подключив его к вашим легальным источникам контента.

## Примечание

Убедитесь, что у вас есть права на трансляцию всех добавляемых файлов.
